# vending-machine
Digital Logic project using Logic Work5

# quick start
Open "vending-machine.cct" via "Logic Work 5"
All done!

# How to use
"vending-machine.cct" consists of 3 Part 
  1. input/output part
  2. state logic part
  3. coputational part
 just play with the 1st part and the machine will not crash:)
